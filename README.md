# 📚 Book Recommender 9001

Use this app to discover new books tailored to your own personal taste. By providing a list of books and genres you enjoy (as well as those you don’t), this app will generate curated recommendations for you.

This app was developed for a class project.

## 🚀 Features

- **Personalized Recommendations**
Generates book suggestions based on user preferences.
- **AI-Powered Engine**
Leverages the OpenAI API to produce relevant and diverse recommendations.
- **Simple Web Interface**
Built with Streamlit for an intuitive and interactive user experience.

### 🛠️ Getting Started

1. Clone the repository

   ```
   $ git clone https://github.com/ryanhua31-blip/book-recommender-9001.git
   $ cd book-recommender-9001
   ```

2. Install the requirements

   ```
   $ pip install -r requirements.txt
   ```
3. Configure API key

   Create a `.env` file
   ```
   API_KEY = [your_api_key_here]
   ```

4. Run the app

   ```
   $ streamlit run streamlit_app.py
   ```
